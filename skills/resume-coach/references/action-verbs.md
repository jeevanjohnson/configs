## How to Pick the Right Verb

1. **What did you actually do?** Did you make something, improve something, lead something, or solve something?
2. **What's the strongest version of that action?** "Made" → "Engineered." "Helped" → "Drove."
3. **Check the job description.** Does it use certain verbs? Mirror that language (ATS signal).
4. **Avoid repetition.** Scan your resume — if you already used "Developed," pick something else next time.

## Resume Example (Varied Verbs)

❌ **WEAK** — Too much repetition:
- Developed a mobile app
- Developed features for the backend
- Developed testing strategies

✅ **STRONG** — Verbs matched to action:
- **Engineered** a mobile app for iOS serving 5K+ daily users
- **Designed** and **implemented** backend features handling 1M+ requests/day
- **Established** testing strategies, increasing code coverage from 40% to 85%
